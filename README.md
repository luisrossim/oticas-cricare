# 🚀 Óticas Cricaré — Vitrine Custo/Manutenção Zero

Uma solução **Full Stack** de alto desempenho, projetada para ser **auto-escalável** e com **custo zero de infraestrutura**. O projeto combina uma vitrine de produtos administrável e auto updated com integrações estratégicas de redes sociais e uma arquitetura baseada em serverless.

O objetivo deste projeto é fornecer às **Óticas Cricaré** uma presença digital robusta que vai além de um site institucional, funcionando como uma vitrine dinâmica de produtos (galeria) com integração direta aos principais canais de venda e conversão.

### Principais Funcionalidades
* **Vitrine & Galeria:** Exibição dinâmica de armações e óculos com suporte a imagens de alta resolução.
* **Painel Administrativo:** Gestão de conteúdo integrada para atualização rápida da vitrine.
* **Autenticação Segura:** Sistema de acesso protegido para gestão da plataforma via Supabase Auth.
* **Omnichannel:** Integração nativa com Instagram, Facebook e WhatsApp para facilitar a jornada de compra.
* **SEO Avançado:** Otimização técnica para mecanismos de busca, focando em visibilidade local e autoridade de marca.

## 🛠️ Stack

O projeto utiliza o conceito de **Composable Architecture**, utilizando os melhores serviços *free-tier* do mercado:

| Camada | Tecnologia | Função |
| :--- | :--- | :--- |
| **Framework** | **Next.js (App Router)** | Renderização híbrida (SSR/Static) e Roteamento. |
| **Database & Auth** | **Supabase** | Banco de dados PostgreSQL e autenticação JWT. |
| **Edge Computing** | **Cloudflare** | DNS, Segurança (WAF) e Cache de borda global. |
| **Cloud Service** | **AWS** | Processamento de tarefas e armazenamento escalável. |
| **Infraestrutura** | **Serverless** | Execução de lógica sob demanda sem custos fixos de servidor. |
